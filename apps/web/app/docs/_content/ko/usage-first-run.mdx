# 최초 설정

<Stepper current={0} locale="ko" />

갓 설치한 허브를 처음 열면 아직 사용자가 없습니다. 이 페이지는 오너 계정 생성과 로그인을 다룹니다. (허브를 먼저 설치하세요: [RelayRoom 설치](/docs/ko/self-hosting).)

## 1. 오너 계정 만들기

`http://localhost:48800`을 엽니다. 사용자가 없으면 `/account/setup`으로 이동하고, 거기서 이름·이메일·비밀번호로 **오너** 계정을 만듭니다. 관리자가 없는 동안에만 가능한 일회성 부트스트랩이며, **메일 서버가 필요 없습니다** - 계정이 바로 생성됩니다.

## 2. 로그인

이후로는 `/account/sign-in`에서 이메일·비밀번호로 로그인합니다. 카드가 가운데 정렬되고, 구석에 테마·언어 스위치가 있습니다.

![로그인](/docs-screens/signin-light.png)

로그인하면 대시보드로 갑니다. 다음으로 프로젝트를 만듭니다.

## 팀원 초대 (나중에)

조직이 생기면 대시보드에서 팀원을 초대합니다. SMTP를 설정했으면 메일이 가고, 없으면 초대 링크가 UI에 표시되고 서버 로그에 출력되니 복사해 전달하면 됩니다. 메일 서버는 선택입니다.

→ 다음: [프로젝트 생성](/docs/ko/projects)
